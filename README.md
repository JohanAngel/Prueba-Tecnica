
# Descripción

Este proyecto es un servicio desarrollado en Spring Boot que gestiona productos, permitiendo realizar operaciones básicas como: 

- crear
- consultar
- actualizar
- eliminar
- filtrar

# Tecnologías principales

- Java 21 
- Spring Boot  
- Maven  
- JUnit y Mockito  

---

# Funcionalidades

- Listar todos los productos  
- Buscar producto por ID  
- Filtrar productos por distintos campos  
- Crear nuevos productos  
- Actualizar información existente  
- Eliminar productos  

---

# Pruebas unitarias

El proyecto incluye pruebas unitarias para validar el correcto funcionamiento de los métodos del servicio.  
Se utilizan JUnit y Mockito para simular el comportamiento del repositorio.

---

# Swagger

Documentación Swagger

El proyecto cuenta con integración de Swagger UI para explorar y probar los endpoints de forma interactiva.
Una vez iniciado el servicio, puedes acceder a la documentación en:

http://localhost:8080/swagger-ui/index.html


Desde esta interfaz podrás visualizar todos los endpoints disponibles, sus métodos HTTP, parámetros y respuestas posibles.

---

La aplicación se ejecutará por defecto en `http://localhost:8080`.
